# Register-files
## PCB DESIGN AND FABRICATION (LM555)

This project involved designing and fabricating a PCB for an LM555-based astable clock circuit. The schematic and layout were created in EasyEDA, including proper component placement, routing, and design rule checks (DRC).

The PCB was then manually fabricated using the toner transfer method, followed by chemical etching (ferric chloride), drilling, and soldering of components.

After assembly, the circuit was tested using an oscilloscope, where a square wave output (~1 Hz) was observed across the LED and a sawtooth waveform across the capacitor, confirming correct operation of the astable multivibrator.


## Cadence Virtuoso – 8-Bit Register Design

Designed and simulated an 8-bit register using Cadence Virtuoso as part of digital IC design work. The project involved schematic design followed by layout creation and full verification flow.

Completed DRC (Design Rule Check), LVS (Layout vs Schematic), PEX (Parasitic Extraction), and post-layout simulation to ensure correctness and performance of the design.

This helped in understanding the complete VLSI design flow from circuit design to layout-level validation.

## Hardware Implementation – 8-Bit Computer Modules

Worked on building and integrating hardware modules of an 8-bit computer on a breadboard. This included designing and testing circuits like registers and clock modules, and ensuring proper interaction between different subsystems such as ALU and control signals.

Faced and resolved practical issues such as wiring faults, incorrect signal flow, and power limitations during integration. This project strengthened hands-on understanding of digital electronics and system-level design.
